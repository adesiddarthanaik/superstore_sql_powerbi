## Superstore Sales Analysis — SQL + Power BI

Analyzed 10,000+ retail transactions using SQL and Power BI Desktop.

### Dashboard Visuals
- KPI Cards: Total Sales (2.30M), Total Profit (286K), Total Orders (5K)
- Sales by Region (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Category Profit Breakdown (Donut Chart)
- Discount vs Profit Analysis (Scatter Chart)

### Key Findings
- West region drives highest revenue
- Technology category has highest profit margins
- Discounts above 20% consistently generate negative profit

### Tools
SQLite · Python · Pandas · Power BI Desktop
